---
title: ⏳ LoadingScreen
description: Padded loading state for full-screen phases.
---

`<LoadingScreen>` wraps `<Spinner>` in a padded container for full-screen loading phases.

```tsx
import { LoadingScreen } from "@kud/ink-ui";
```

## 🔧 Props

| Prop    | Type     | Description              |
| ------- | -------- | ------------------------ |
| `label` | `string` | Spinner label to display |

## 💡 Examples

```tsx
<LoadingScreen label="Fetching PR…" />
```
