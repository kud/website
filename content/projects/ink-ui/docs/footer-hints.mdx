---
title: ⌨️ FooterHints
description: Wrapped keyboard shortcut hints for interactive screens.
---

`<FooterHints>` renders consistent keyboard shortcut help for interactive terminal screens.

```tsx
import { FooterHints } from "@kud/ink-ui";
import type { Hint } from "@kud/ink-ui";
```

## 🔧 Props

| Prop    | Type     | Description                     |
| ------- | -------- | ------------------------------- |
| `hints` | `Hint[]` | Tuples of key label and meaning |

## 💡 Examples

```tsx
<FooterHints
  hints={[
    ["↑↓", "navigate"],
    ["↵", "select"],
    ["q", "quit"],
  ]}
/>
```
