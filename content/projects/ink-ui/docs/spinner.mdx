---
title: Spinner
description: Animated dots spinner in accent colour, with an optional dimmed label.
---

`<Spinner>` renders an animated dots spinner in the accent colour, with an optional dimmed label beside it. It builds on [ink-spinner](https://github.com/vadimdemedes/ink-spinner) and is ideal for indicating in-progress work.

```tsx
import { Spinner } from "@kud/ink-ui";
```

## Props

| Prop    | Type     | Default | Description                               |
| ------- | -------- | ------- | ----------------------------------------- |
| `label` | `string` | —       | Optional text rendered beside the spinner |

The spinner glyph is coloured with `colors.accent` (`#FF8C00`).

## Examples

```tsx
<Spinner label="Building…" />
```

```tsx
<Spinner />
```
