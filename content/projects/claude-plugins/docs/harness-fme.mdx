The **harness-fme** plugin wraps the
[`@kud/mcp-harness-fme`](https://www.npmjs.com/package/@kud/mcp-harness-fme) MCP
server. Inspect and control Harness FME feature flags: list environments, audit
targeting rules, and kill or restore flags in a single conversation.

## What it bundles

| Skill                  | Trigger                       | What it does                                          |
| ---------------------- | ----------------------------- | ----------------------------------------------------- |
| `/feature-flag-status` | `/feature-flag-status <flag>` | Full status report for a flag across all environments |

## Install

```
/plugin install harness-fme@kud --scope project
```

Project scope is recommended so a whole team shares the same Harness config via
`.claude/settings.json`.

## Configuration

Provide your Harness FME API key:

```
MCP_HARNESS_FME_API_KEY=your-api-key
```
