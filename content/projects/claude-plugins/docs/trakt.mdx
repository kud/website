The **trakt** plugin wraps the
[`@kud/mcp-trakt`](https://www.npmjs.com/package/@kud/mcp-trakt) MCP server.
Track what you're watching: search movies and shows, check in, browse your
history, manage your watchlist, and get recommendations.

## What it bundles

| Skill             | Trigger                  | What it does                                             |
| ----------------- | ------------------------ | -------------------------------------------------------- |
| `/trakt-whats-on` | `/trakt-whats-on`        | See your watchlist, recently watched, and what's up next |
| `/trakt-checkin`  | `/trakt-checkin <title>` | Check in to a movie or episode you're about to watch     |

## Install

```
/plugin install trakt@kud --scope user
```

## Configuration

Authenticate with Trakt once via the bundled setup command:

```
npx @kud/mcp-trakt@latest setup
```
