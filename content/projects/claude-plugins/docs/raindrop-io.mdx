The **raindrop-io** plugin wraps the
[`@kud/mcp-raindrop-io`](https://www.npmjs.com/package/@kud/mcp-raindrop-io) MCP
server. Manage your Raindrop.io bookmarks from Claude: search your library, save
new links, organise into collections, and tag.

## What it bundles

| Skill              | Trigger                    | What it does                                         |
| ------------------ | -------------------------- | ---------------------------------------------------- |
| `/bookmark-search` | `/bookmark-search <query>` | Search your bookmarks by keyword, tag, or collection |
| `/bookmark-save`   | `/bookmark-save <url>`     | Save a URL to Raindrop.io with tags and collection   |

## Install

```
/plugin install raindrop-io@kud --scope user
```

## Configuration

Provide your Raindrop.io API token:

```
MCP_RAINDROPIO_TOKEN=your-token
```
