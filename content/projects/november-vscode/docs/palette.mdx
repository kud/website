## Interface

These are the key workbench colours read from the theme.

| Role               | Hex         |
| ------------------ | ----------- |
| Background         | `#21222c`   |
| Foreground         | `#BFC7D5`   |
| Accent (cursor)    | `#ed7249`   |
| Selection          | `#ed724940` |
| Line number        | `#4c5374`   |
| Active line number | `#eeffff`   |

## Syntax tokens

The main syntax token colours used across languages.

| Token                 | Hex       |
| --------------------- | --------- |
| Comment               | `#697098` |
| String                | `#C3E88D` |
| Number                | `#F78C6C` |
| Constant / boolean    | `#ff5874` |
| Variable              | `#ffcb6b` |
| Keyword / storage     | `#c792ea` |
| Class / type          | `#ffcb6b` |
| Function name         | `#82AAFF` |
| Operator              | `#89DDFF` |
| Tag name              | `#ff5572` |
| RegExp / raw code     | `#80CBC4` |
| Markup heading / link | `#82b1ff` |
