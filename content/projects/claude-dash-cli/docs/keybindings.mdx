---
title: Keybindings
description: The full keyboard reference for navigating sessions, handling permissions, and managing the dashboard.
---

Every action in claude-dash is a single keypress. Navigate the session list,
respond to permission prompts, and manage sessions without ever reaching for the
mouse.

| Key       | Action                                                                         |
| --------- | ------------------------------------------------------------------------------ |
| `↑` / `k` | Select previous session                                                        |
| `↓` / `j` | Select next session                                                            |
| `a`       | Allow pending permission                                                       |
| `s`       | Allow permission for this session (auto-approve future requests for same tool) |
| `d`       | Deny pending permission                                                        |
| `e`       | Rename selected session                                                        |
| `n`       | Launch a new Claude session                                                    |
| `r`       | Refresh usage data                                                             |
| `x`       | Clear ended sessions                                                           |
| `q`       | Quit                                                                           |
| `Q`       | Quit and kill daemon                                                           |

The agent list also supports mouse-wheel scrolling.
