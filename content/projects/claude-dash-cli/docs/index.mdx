**claude-dash** is a terminal dashboard for monitoring your
[Claude Code](https://claude.ai/code) sessions in real time. See every running
agent at a glance, approve permission prompts without leaving the terminal, and
track your usage and spend live.

Built entirely in Rust with [Ratatui](https://ratatui.rs). Zero lag, no
Electron, no browser, no Node.js — just a single binary.

## Install

```bash
cargo install claude-dash
claude-dash install
```

`cargo install` builds and installs the `claude-dash` binary from
[crates.io](https://crates.io/crates/claude-dash). The `claude-dash install`
step wires up the Claude Code hooks (writes `~/.claude/settings.json`) so the
dashboard can track your sessions.

Then just run:

```bash
claude-dash
```

The TUI launches immediately and auto-spawns its background daemon on first run.

## Requirements

- macOS (Linux should work for the TUI; `osascript` session launch only works on
  macOS)
- [Claude Code](https://claude.ai/code) installed
- A [Nerd Font](https://www.nerdfonts.com) terminal font for icons (optional but
  recommended)

Head to [Usage](/projects/claude-dash-cli/docs/usage) to get going, or skim the
[Features](/projects/claude-dash-cli/docs/features).
