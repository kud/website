## Install

```bash
npm install -g @kud/pcloud-cli
```

## Development

Run directly from source without a build step:

```bash
npm run dev -- list-trash
npm run dev -- list-rewind /some/path
```

Build compiled output to `dist/`:

```bash
npm run build
```
