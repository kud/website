mcp-qobuz is a [Model Context Protocol](https://modelcontextprotocol.io) server that exposes Qobuz to AI clients like Claude Desktop and Claude Code — so an assistant can search the catalogue, look up tracks and albums, browse your favourites, see what's now playing, and manage playlists on your behalf.

It is the AI-facing surface of the `@kud/qobuz` ecosystem, sharing the same core as the CLI and the Raycast extension.
