---
title: Image Attachments
description: Attach files or inline base64 images to a query for vision-capable models.
---

The `query` tool accepts an optional `attachments` array for vision-capable
models (Claude, GPT-5+).

## File attachment

Reads from disk:

```json
{
  "type": "file",
  "path": "/absolute/path/to/screenshot.png"
}
```

## Blob attachment

Inline base64 data:

```json
{
  "type": "blob",
  "data": "<base64-encoded-content>",
  "mimeType": "image/png"
}
```

Both attachment types support an optional `displayName` field.
