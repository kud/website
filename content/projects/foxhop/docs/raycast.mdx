Both Raycast surfaces are driven by the same `tabs.json`.

## Fox Hop extension

A single **Focus Tab** command to search, focus, add, edit, and delete targets, plus a **Generate Hotkey Scripts** command that runs `foxhop sync` for you. Install it from the Raycast Store (or run it from source).

## Per-tab hotkey scripts

`foxhop sync` writes one Raycast Script Command per target into `~/.config/foxhop/scripts`. Add that folder in Raycast → Extensions → Script Commands → Add Directories, then assign a global hotkey to each. Re-run `foxhop sync` after editing your tabs — each generated script carries the foxhop icon.
