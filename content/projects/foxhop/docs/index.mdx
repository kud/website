foxhop focuses a specific Firefox tab — your ChatGPT, Todoist, email, a dashboard — from anywhere on macOS, via a global hotkey, Raycast, or the terminal. By default it does nothing: you decide which tabs matter by listing them in `~/.config/foxhop/tabs.json`.
