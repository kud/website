## The extension must be loaded

foxhop does nothing until the Firefox extension is loaded and connected. Without it, `foxhop focus` falls back to _opening_ the URL rather than focusing the existing tab.

## Firefox Nightly is the default

`npm run dev` uses `--firefox=nightly`. Temporary add-ons (loaded via `about:debugging`) must be **reloaded** after changing `extension/src/background.js`, and they unload when you restart Firefox. For a permanent install, install the signed `.xpi` from addons.mozilla.org.

## Moving the repo

The native-messaging manifest's `path` is absolute — if you move the repo (when running from source), re-run `foxhop install`. Generated hotkey scripts embed the mise Node shim, so they survive Node upgrades.
