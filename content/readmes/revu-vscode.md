**Annotate any line of code. Export your review to an AI chat client.**

Built for the AI-assisted coding workflow — review what the AI wrote, annotate what needs fixing, and send it back as a single prompt-led Markdown review.

## Install

Search **revu** in the VS Code Extensions panel, or:

```sh
code --install-extension kud.revu-vscode
```

## Features

- Annotate any line or selection as inline comment threads, with a sidebar panel and Markdown export to Copilot, Claude Code, the clipboard, or a Markdown preview.
