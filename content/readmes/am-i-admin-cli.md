> A tiny macOS CLI that tells you whether you're an admin on this computer.

## Install

```sh
npm install -g @kud/am-i-admin-cli
```

## Usage

```sh
am-i-admin          # human-readable verdict
am-i-admin --raw    # bare boolean: true / false
```

## Development

```sh
npm install
node index.js
```
