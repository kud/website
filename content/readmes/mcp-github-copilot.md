## Install

```bash
npm install -g @kud/mcp-github-copilot
```

## Development

```bash
git clone https://github.com/kud/mcp-github-copilot.git
cd mcp-github-copilot
npm install
npm run build
npm test
```

Use the MCP Inspector to test tools interactively:

```bash
npm run inspect
# Opens http://localhost:5173
```
