Claude Code plugins bundle an **MCP server** + **companion skills** into a single installable unit. Each plugin here wraps one of kud's published MCP servers and ships ready-to-use slash commands for the most common workflows. Install a plugin → get the MCP tools _and_ the skills in one shot.

## 🚀 Install

```
/plugin marketplace add kud/claude-plugins  # registers as @kud
```
