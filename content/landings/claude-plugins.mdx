<Section
  eyebrow="Why claude-plugins"
  title="The integration and its skills, in one install"
  sub="Normally you register an MCP server or CLI workflow, edit config files, then hunt down the matching skills and install them separately. Each plugin here bundles the integration with its companion skills into a single unit — install one, get the tools and the workflows in one shot."
>
  <Install label="the usual way">

```text
# 1. add the integration to your config
# 2. find and install the matching skills
# 3. wire up the credentials by hand
```

  </Install>

  <Install label="with claude-plugins">

```text
/plugin install opencode@kud
```

  </Install>
</Section>

<Section eyebrow="Quick start" title="Up and running in one command">
  <Install label="1 · register the marketplace">

```text
/plugin marketplace add kud/claude-plugins
```

  </Install>

  <Install label="2 · install a plugin">

```text
/plugin install opencode@kud --scope user
```

  </Install>

  <Install label="3 · use a skill">

```text
/ask-opencode explain this function
```

  </Install>
</Section>

<Section
  eyebrow="The plugins"
  title="Eight integrations, ready to install"
  sub="Each plugin wraps one of kud's published MCP servers or CLI workflows and ships the slash commands for its most common tasks."
>
  <Features>
    <Feature title="opencode">
      Query any opencode-supported model for a second opinion, right inside
      Claude Code.
    </Feature>
    <Feature title="github-copilot">
      Query GitHub Copilot models through the official SDK.
    </Feature>
    <Feature title="jenkins">
      Inspect builds, stream logs, and diagnose CI failures without leaving the
      terminal.
    </Feature>
    <Feature title="harness-fme">
      Inspect and control Harness FME feature flags.
    </Feature>
    <Feature title="trakt">
      Track what you're watching: check in, manage your watchlist, browse
      history.
    </Feature>
    <Feature title="raindrop-io">
      Search and save Raindrop.io bookmarks.
    </Feature>
    <Feature title="google-keep">
      Capture notes and manage Google Keep checklists.
    </Feature>
    <Feature title="gandi">
      Manage Gandi domains, DNS records, and web redirects through the
      @kud/gandi-cli workflow.
    </Feature>
  </Features>
</Section>
